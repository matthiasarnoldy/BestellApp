# BestellAPP

Diese Applikation ermöglicht es Ihnen, schnell und einfach essen zu bestellen.


## Verwendung

1. Artikel zum Warenkorb hinzufügen
- Gericht auswählen und das '+'-Symbol betätigen (kann durch ernuetes Betätigen wieder zum Warenkorb hinzugefügt werden)
2. Artikel aus dem Warenkorb entfernen
- Über das Papierkorb-Symbol kann das Gericht wieder aus dem Warenkorb entfernt werden
- falls kein Papierkorb-Symbol vorhanden ist, über das '-'-Symbol die Anzahl des Gerichtes verringern
3. Bestellung aufgeben
- Über den Button Bestellen (Preis der Bestellung) kann die Bestellung aufgegeben werden- falls der Warenkorb leer sein sollte, wird eine Fehlermeldung ausgegeben, dass sich im Warenkorb nichts befindet
